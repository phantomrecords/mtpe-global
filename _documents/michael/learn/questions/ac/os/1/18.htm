What does placing BUFFERS=20 in CONFIG.SYS do?
allocate program-buffering memory for programs to use
allocate disk-buffering memory for programs to use
allocate memory that is isolated from system resources to improve troubleshooting attempts
define a virtual memory region for the operating system

Jennifer Flynn
10 Minute Guide To Memory Management
Pg. 35 ¶ One  
  “The BUFFERS statement sets the number of disk buffers.  Disk buffers create a cache in RAM, containing information recently read from disk.  When a request for data occurs, a quick scan of the disk buffer is made.  If the requested data is in the disk buffer, then the data is transferred from the buffer into the program's working area in RAM.  Because the transfer is between two areas of RAM, this process is very fast.”
“A complex database program may require 20 or even 30 buffers; a word processor, probably requires only 10 to 15.”
